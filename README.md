# Api Sentry

Api Sentry is a Python project that provides a simple API for tracking and analyzing performance metrics for API endpoints.

## Usage

To use Api Sentry, create an instance of the `ApiSentry` class and add performance metrics using the `add_metric` method. You can then retrieve historical data for an endpoint using the `get_historical_data` method, or get trend analysis using the `get_trend_analysis` method.

## Testing

To run the tests, use the `pytest` command.
